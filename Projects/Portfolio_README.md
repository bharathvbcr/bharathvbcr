## üöÄ Website Features

This portfolio is built as a modern, high-performance Single Page Application (SPA) with a focus on interactive design and user experience.

- **Interactive 3D Hero Section:** A dynamic, interactive 3D cell model built with **Three.js** that responds to mouse movements.
- **Hexagon Skills Grid:** A futuristic, sci-fi inspired honeycomb layout for displaying technical skills with glowing effects.
- **Interactive Modals:** Detailed views for projects and skills, featuring image carousels and circular proficiency indicators.
- **Modern UI/UX:** Designed with **Tailwind CSS** for a sleek, dark-themed, and fully responsive experience.
- **Glassmorphism Design:** Uses backdrop blur and semi-transparent elements for a premium feel.
- **Firebase Integration:** Real-time data fetching for projects and a working contact form connected to Firestore.
- **SEO Optimized:** Includes dynamic meta tags, sitemap, and structured data for better visibility.

To run this project locally:

1. **Clone the repository:**

    ```bash
    git clone https://github.com/bharathvbcr/my-portfolio.git
    cd my-portfolio
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Start the development server:**

    ```bash
    npm run dev
    ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.
